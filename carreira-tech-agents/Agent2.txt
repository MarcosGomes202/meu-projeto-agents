💬 ROTEIRO DE ENTREVISTAS

PERGUNTA 1: O que é regressão linear?
COMO RESPONDER: É um modelo estatístico usado para prever valores contínuos com base em variáveis independentes.

PERGUNTA 2: Qual a diferença entre dados estruturados e não estruturados?
COMO RESPONDER: Estruturados são organizados em tabelas (SQL), não estruturados são textos, imagens, vídeos.

PERGUNTA 3: Como você lida com dados faltantes?
COMO RESPONDER: Técnicas como remoção de linhas, imputação de valores médios ou uso de algoritmos que lidam com missing values.

PERGUNTA 4: Qual biblioteca Python você usaria para machine learning?
COMO RESPONDER: Scikit-learn para modelos básicos, TensorFlow ou PyTorch para modelos avançados.

PERGUNTA 5: Conte sobre um projeto que você fez com dados.
COMO RESPONDER: Descreva o projeto de portfólio, destacando problema, solução e resultados.
